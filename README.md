# Online Courier Tracking System

The Online Courier Tracking System is a web-based application that allows users to track courier parcels using a unique tracking ID. The system helps manage courier bookings, shipment status, dispatch details, and delivery updates. It provides separate access for admin and users to monitor and update parcel information efficiently.

Technologies used: HTML, CSS, Bootstrap, Java/Spring Boot, and MySQL.
